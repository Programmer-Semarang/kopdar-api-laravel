# kopdar-api-laravel
Kopdar Programmer Semarang tema API with Laravel

## How to Install

* Clone this repository
* Install Dependencies
```bash
composer install
```
* Copy .env
```
cp .env.example .env
```
* Generate app key
```
php artisan key:generate
```
* Add write permission (Unix)
```
chmod -R go+w storage bootstrap/cache
```
